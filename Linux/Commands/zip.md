#linux #command #zip
## zip
file packaging utility compression technique

`zip  <zipFileName> fileDir/fileList`

- **-u** : add more files to existing zipfile
- **-d** : remove files to existing zipfile
- **-m** : move files into zip by deleting original
- **-x** : exclude files
- **-r** : recursive
- **-v** : verbose
- **-e** : encrypt
